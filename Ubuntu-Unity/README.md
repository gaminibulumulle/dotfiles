# Configuring `dwm`

> Configuring `DWM` is easy as 123. 

```shell
cd dwm
sudo make install
```

```shell
cd st
sudo make install
```

```shell
cd slstatus
sudo make install
```


# Notes:

> In Ubuntu, by default, `git` is not installed. To install git, 
> please type this command:

> `sudo apt install git`
