Windows Configurations Files 

Introduction: 
--------------------

In this section, we are going to be talking about the Windows Configurations to be used only 
on Windows. Normally Senal's configuration files are compatible to both Unix/Linux, so this
particular one is special for use only on Windows. 




Overview
------------------------

NeoVim Configurations
-------------------------
Inside this configuration file, this neovim configuration file allows the abilty to use 
some of Senal's neovim configuration file that he uses as a daily. Such as the vim plugin
and setups from scratch. Normally the script installs into this directory here: 

[C\Users\{user}\AppData\Local\nvim]
