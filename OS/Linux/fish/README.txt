					Welcome to Senal's Fish Config. This documentation
					shows you how to use Senal's fish configurations. 


					Aliases: 

					Aliases are shortcuts that you can configure when 
					you are using the terminal making the workflow much
					more efficient. 


					Here are some aliases that Senal uses. 


					Alias: detect-model

					If the user types in 'detect-model', it will prompt 
					the user the superuser password, before showing the 
					model name, and the firmware version of the hardware.
